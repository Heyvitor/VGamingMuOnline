# Repositorio VGaming

Este repositorio centraliza codigos, projetos e materiais que serao publicados nos canais da comunidade VGaming. 
A proposta e manter aqui os arquivos que serao compartilhados com a comunidade de Mu Online e com o projeto Open Source em construcao.

## Discords

- Comunidade VGaming MuOnline: https://discord.gg/maGp7u7a
- Mu Online CrossPlataform Open Source em construcao: https://discord.gg/HevQ5dK5

## Objetivo

Este repositorio sera usado para armazenar codigos que serao liberados no Discord, organizando o material em um unico lugar para facilitar consulta, manutencao e distribuicao.

## Estrutura Atual

- `EditorBMD/`: arquivos do editor de BMD do 5.2 Takumi.
- `Docker/`: Imagens Docker.
